## Design Notes

### Colors
- Primary: Blue (#2563eb)
- Success/Income: Green (#16a34a)
- Expense: Red (#dc2626)
- Warning: Amber (#d97706)
- Background: Light gray (#f3f4f6)
- Card background: White

### Touch Targets
- Minimum 44x44px for all interactive elements
- Adequate spacing between tap targets

### Typography
- System fonts (fast loading)
- 16px base size (no zoom on mobile inputs)
- Clear hierarchy with size/weight

### Responsive Breakpoints
- Mobile: 360px+
- Tablet: 768px+
- Desktop: 1024px+

The app is mobile-first; desktop just gets more breathing room.
